MialaPlay - Musique & Videos

Pour faire cela, il existe deux options après avoir choisi votre playlist puis actualiser:

  1. Procédure standard: Veuillez suivres ces étapes:
    1. Copiez ce lien et mettez-le en favoris (Une bonne fois pour toute);
    2. Actualiser la page de votre playlist YouTube;
    3. Sur la page de la playlist, cliquez sur votre favoris;
    4. Appuyez sur le bouton 'OK';
    5. Appuyez sur le bouton 'Scanner la playlist';
    6. Patientez jusqu'à 100% (dans le titre)...
    7. Lecture en mode aléatoire ? OK=Oui, Annuler=Non
    8. Et voilà ! Il vous suffit de démarer la première vidéo !

  2. Procédure exceptionnelle: Copiez le code ci-dessous puis rentrez-le dans la console (dans inspecter). Suivez ensuite le tuto à partir du 2.
